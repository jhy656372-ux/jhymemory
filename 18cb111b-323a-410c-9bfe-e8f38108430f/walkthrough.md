# 산돌교회 홈페이지 제작 완료

사용자(영자야)의 요청에 따라 산돌교회 홈페이지의 첫 페이지를 성공적으로 구현했습니다.
모던하고 깔끔한 디자인을 적용하였으며, 요청하신 모든 섹션을 포함했습니다.

## 주요 기능 및 특징
- **반응형 디자인**: 모바일, 태블릿, 데스크탑 등 모든 기기에서 최적화된 화면을 제공합니다.
- **부드러운 스크롤**: 네비게이션 메뉴 클릭 시 해당 섹션으로 부드럽게 이동합니다.
- **애니메이션 효과**: Framer Motion을 사용하여 스크롤 시 요소들이 자연스럽게 나타나는 효과를 적용했습니다.
- **카드형 레이아웃**: 주일학교, 전도회, 지역섬김 섹션에 통일된 카드 디자인을 적용하여 가독성을 높였습니다.
- **섹션 구성**:
    1.  **헤더**: 스크롤에 따라 변하는 투명/화이트 네비게이션 바
    2.  **히어로**: 웅장한 배경 이미지와 환영 메시지 (버튼 제거됨)
    3.  **데이터 분리**: `src/data/churchData.js` 파일에서 모든 텍스트와 이미지 관리
    4.  **수양관 페이지**: 시설 안내 인터랙티브 기능(클릭 시 이미지 변경) 및 깔끔한 레이아웃 적용
    5.  **상세 페이지**: 각 섹션(교회소개, 주일학교 등)을 클릭하면 자세한 내용을 담은 별도 페이지로 이동합니다.
    6.  **푸터**: 전체 너비에 맞춘 박스형 디자인으로 수정하여 통일감 부여
    7.  **관리자 모드**: `/admin` 페이지에서 데이터 시각적 편집 및 이미지 업로드 기능 제공

## 데이터 관리 (Content Management)

### 1. 관리자 페이지 (쉬운 방법)
홈페이지 맨 하단의 `Admin` 글씨를 클릭하거나 주소창 뒤에 `/admin`을 붙여 관리자 페이지로 이동하세요.
*   **수양관 관리**: 텍스트를 직접 수정하고, **[파일 선택]** 버튼을 눌러 컴퓨터의 사진을 업로드하면 즉시 반영됩니다.
*   *주의: 이 변경사항은 귀하의 브라우저에만 저장됩니다.*

### 2. 코드 수정 (영구적인 방법)
`src/data/churchData.js` 파일을 직접 수정하면 모든 방문자에게 변경된 내용이 적용됩니다.
홈페이지의 모든 텍스트와 이미지는 **`src/data/churchData.js`** 파일에서 통합 관리됩니다.
이 파일의 내용만 수정하면 홈페이지 전체에 즉시 반영되므로, 개발 지식이 없어도 쉽게 내용을 업데이트할 수 있습니다.

### 🖼️ 수양관 사진 바꾸는 방법
`src/data/churchData.js` 파일을 열고 `retreat` 부분을 찾으세요. 원하시는 사진의 인터넷 주소(URL)를 `image` 옆의 따옴표(`""`) 안에 넣어주시면 됩니다.

```javascript
features: [
    {
        name: "예배실 및 소그룹 모임방 완비",
        // 여기 아래의 주소를 바꾸시면 사진이 바뀝니다!
        image: "https://새로운_사진_주소.jpg"
    },
    // ...
]
```

## 실행 방법

터미널에서 다음 명령어를 입력하여 홈페이지를 실행할 수 있습니다.

```bash
cd "c:/Users/admin/.gemini/산돌교회 홈페이지"
npm install
npm run dev
```
