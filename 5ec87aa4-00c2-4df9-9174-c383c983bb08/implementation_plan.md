## Proposed Changes

### Frontend - React App
#### [MODIFY] `src/App.jsx`
- 사용되지 않는 `marketIndices` 상태 및 데이터를 불러오는 폴링 타이머 제거 (네트워크 리소스 낭비 방지)
- 주식 상세 데이터(`stockData`)의 실시간 업데이트 간격을 기존 15초에서 5초로 줄여 실시간 체감 성능 향상
- 메인 페이지 하단 명언 문구 보존 확인

#### [MODIFY] `src/services/StockService.js`
- **시간대 분리 엔진**: 한국 주식(09:00~15:30)과 미국 주식(09:30~16:00)의 장중 시간분을 정확하게(390분) 10분 단위로 세그먼트하여(count: 40) 시간 차트에 노출시키는 엔진 명시화.
- **정밀 수식 차트 생성기**:
  - `Math.random()` 의존성을 제거하고 Seed 기반의 Deterministic한 파동 산출
  - 파동의 마지막 데이터 단가와 실제 실시간 `actualPrice` 간의 차이(`diff`)를 구함
  - 이 `diff` 오차분을 차트 0번 인덱스부터 마지막 인덱스까지 비율에 맞춰 누적 더해주는 보정 수식을 적용
  - 결과적으로 차트가 중간에 튀지 않고 정확하게 **100% 실시간 현재가로 유려하게 수렴**
