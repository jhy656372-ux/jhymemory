아름답고 은혜로운 분위기의 현대적인 교회 웹사이트를 구축하고, 사용자님의 요청에 따라 **로컬 AI(@local, `http://localhost:11434/v1`)**를 활용한 스마트한 기능을 포함하는 계획입니다.

## User Review Required

> [!IMPORTANT]
> 로컬 AI를 활용하는 구체적인 기능에 대해 원하시는 방향이 있으신지 확인 부탁드립니다.
> 현재는 **'AI 신앙 질문 챗봇(AI 목사님)'** 또는 사용자의 기분/상황을 입력하면 **'맞춤형 성경 구절과 기도문을 생성해주는 기능'**을 제안합니다. 마음에 드시는 방향을 선택해 주세요!

> [!WARNING]
> 로컬 AI(Ollama)와 웹 브라우저가 직접 통신할 때 CORS(교차 출처 리소스 공유) 에러가 발생할 수 있습니다. 이를 방지하기 위해 웹 애플리케이션 개발용 서버(Vite)에 Proxy를 설정하여 안전하게 통신하도록 구성할 예정입니다.

## Proposed Changes

`C:\Users\admin\.gemini\antigravity\scratch\church-website` 경로에 새로운 프로젝트를 생성합니다. 최신 프론트엔드 환경인 React와 Vite를 사용하여 빠르고 부드러운 웹사이트를 구축합니다. CSS는 순수 CSS(Vanilla CSS)를 사용하여 세밀하고 고급스러운 애니메이션과 디자인을 적용합니다.

### 1. 프로젝트 초기화 및 환경 설정 (Vite + React)
- 프론트엔드 프로젝트 생성 (`npx create-vite`)
- 로컬 AI API 통신을 위한 Vite Proxy 설정 (`vite.config.js`)

### 2. UI/UX 디자인 및 컴포넌트 개발
- **Hero 섹션**: 환영 인사말과 시각적으로 뛰어난 교회 배경 이미지
- **예배 안내 섹션**: 예배 시간 및 오시는 길 안내
- **로컬 AI 통합 섹션 (@local)**:
  - 사용자가 질문이나 상황을 입력하면 로컬 AI가 답변이나 위로의 성경 구절을 제공하는 아름다운 UI 개발

### 3. 로컬 AI 연동 로직
- `baseUrl: "/api/v1"` (Vite 프록시를 통해 `http://localhost:11434/v1`로 연결)를 활용한 API 통신 로직 작성
- Ollama에서 실행 중인 모델과 통신
