# 데크 제품 쇼핑몰 - 프로젝트 실행 가이드

## 프로젝트 개요
네온/사이버펑크 스타일의 데크(보드) 제품 쇼핑몰입니다. Vite+React 기반으로 설계되었으나, Node.js 환경이 없는 사용자를 위해 **단일 파일 실행 버전**으로 전환되었습니다.

## 주요 기능
- **디자인 시스템**: 다크 모드 기반의 네온/비비드 컬러, 호버 애니메이션
- **페이지 구성**:
  - `Home`: 히어로 섹션, 추천 상품
  - `Shop`: 전체 상품 목록, 카테고리 필터, **실시간 검색**
  - `ProductDetail`: 상품 상세 정보, **장바구니 담기**, **좋아요(찜하기)**
- **User Experience**:
  - **장바구니 모달**: 페이지 이동 없이 상품 목록 확인 가능
  - **단일 파일 실행**: 설치 과정 없이 HTML 파일 하나로 전체 기능 작동

## 실행 방법 (간편 버전)

복잡한 설치 과정 없이 바로 실행할 수 있습니다.

1. 파일 탐색기에서 프로젝트 폴더를 엽니다: `c:\Users\admin\.gemini\테스트7`
2. **`deck_shop_standalone.html`** 파일을 찾습니다.
3. 파일을 더블 클릭하여 실행합니다. (Chrome, Edge 등 브라우저 자동 연결)

## (참고) 개발 모드 실행 방법
Node.js가 설치된 환경에서 원본 코드를 수정하려면 아래 단계를 따르세요.

1. `npm install`
2. `npm run dev`
