# 데크 제품 쇼핑몰 구현 계획서

## 목표
사용자에게 강력한 시각적 인상을 주는 "데크 제품(스케이트보드 등)" 쇼핑몰 웹사이트를 구축합니다.
한국어로 제공되며, 반응형 디자인과 현대적인 UI/UX를 목표로 합니다.

## 사용자 검토 필요 사항
- **디자인 컨셉**: "데크"라는 주제에 맞춰 스케이트보드/스트릿 문화를 반영한 다크 모드 기반의 네온/비비드 컬러 디자인을 제안합니다. 이에 동의하시나요?
- **기술 스택**: Vite + React + Vanilla CSS를 사용하여 빠르고 가볍게 구축하겠습니다. (TailwindCSS는 요청 시에만 사용)

## 변경 제안

### 프로젝트 설정
- [NEW] Vite + React 프로젝트 생성 (TypeScript 아님, 빠른 프로토타이핑을 위해 JavaScript 사용)
- [NEW] `react-router-dom` 설치 (페이지 라우팅)

### 디자인 시스템 (Vanilla CSS)
- `index.css`: CSS 변수를 활용한 컬러 팔레트, 타이포그래피, 리셋 스타일 정의
- **컬러 팔레트**:
  - 배경: 짙은 차콜/블랙 (`#121212`)
  - 포인트: 네온 라임 (`#ccff00`) 또는 일렉트릭 퍼플 (`#bc13fe`)
  - 텍스트: 오프 화이트 (`#f0f0f0`)

### 주요 컴포넌트 구조
- `src/components/`
  - [NEW] `Header.jsx`: 로고, 네비게이션, 장바구니 아이콘
  - [NEW] `Footer.jsx`: 저작권 및 링크
  - [NEW] `ProductCard.jsx`: 상품 이미지와 가격, 호버 효과
  - [NEW] `HeroSection.jsx`: 메인 페이지의 대형 비주얼/비디오 배너
  - [NEW] `CartModal.jsx`: 장바구니 미리보기 및 결제 버튼

### 페이지 구조
- `src/pages/`
  - [NEW] `Home.jsx`: 히어로 섹션 + 추천 상품 큐레이션
  - [NEW] `Shop.jsx`: 전체 상품 목록 (필터링 예정)
  - [NEW] `ProductDetail.jsx`: 상품 상세 정보 및 장바구니 담기

### 데이터 모델
- `src/data/products.js`: 데크 제품 더미 데이터 (이미지 URL 포함)

## 검증 계획

### 자동화 테스트
- 현재 단계에서는 별도의 유닛 테스트보다 빠른 UI 구축에 집중합니다.

### 수동 검증
1. **메인 페이지 로딩**: 히어로 섹션의 애니메이션 및 레이아웃 확인
2. **반응형 동작**: 모바일/데스크탑 화면 크기 조절 시 레이아웃 깨짐 확인
3. **쇼핑 흐름**: 상품 목록 -> 상세 페이지 -> 장바구니 담기 -> 장바구니 확인 흐름이 정상 작동하는지 클릭 테스트
