# 서버 실행 및 환경 복구 계획

사용자님께서 겪고 계신 "서버 응답 없음" 및 "FastAPI 백엔드 확인" 문제는 현재 백엔드 서버가 실행 중이지 않거나, 서버가 사용하는 가상 드라이브(T:)가 연결되어 있지 않아 발생하고 있습니다.

## 제안하는 변경 사항

### 1. 가상 드라이브(T:) 연결
본 프로젝트는 한글 경로 문제(Unicode) 및 긴 경로 문제를 해결하기 위해 전체 프로젝트 폴더를 `T:` 드라이브로 매핑하여 사용하도록 설계되어 있습니다. 현재 `T:` 드라이브 연결이 끊어져 있어 이를 다시 연결하겠습니다.

### 2. 서버 실행 스크립트 작성
매번 복잡한 명령어를 입력할 필요 없이, 더블 클릭 한 번으로 서버를 켤 수 있는 `서버_실행.bat` 파일을 프로젝트 루트 폴더에 생성하겠습니다.

### 3. 서버 상태 점검 및 실행
가상 환경(venv)을 통해 FastAPI 백엔드 서버를 직접 실행하여 정상 작동 여부를 확인합니다.
