# 이미지 립싱크 자연스럽게 개선 결과

인공지능(MediaPipe)을 적용해 더 자연스러운 입모양을 만들도록 립싱크 알고리즘을 성공적으로 개선했습니다.

## 구현된 주요 변경 사항

### 1. `backend/main.py` 리팩토링 완료
- 기존에 사용하던 **OpenCV HAARCASCADE (입술 위치를 대략적으로 추측하는 방식)** 코드를 완전히 지우고, **MediaPipe Face Mesh** 모델로 교체했습니다.
- 이제 478개의 얼굴 관절점(랜드마크) 중 실제 윗입술과 아랫입술, 입꼬리의 정확한 픽셀 위치를 계산하여 입술의 움직임을 제어합니다.

### 2. 자연스러운 하관(턱밑) 움직임 연동
- 입술을 기준으로 윗얼굴은 가만히 두고, 아랫입술 아래쪽의 **하관 부분만 오디오 볼륨에 맞춰 움직이도록(Warping)** 마스킹 처리를 세밀하게 조정했습니다.
- 입술이 벌어질 때 기존 이미지의 픽셀(보통 어두운 입안 색상)이 자연스럽게 늘어나며 실제로 입을 벌리는 듯한 효과를 줍니다.

### 3. 입술 안쪽(Mouth Interior) 자동 생성 기술 추가
- 오디오 볼륨에 의해 턱이 아래로 벌어질 때 기존 이미지가 늘어나기만 하는 현상을 보완했습니다.
- MediaPipe를 통해 위아래 입술 안쪽 라인(Inner Lip Polygon)을 정확히 분리하고, 입이 벌어진 틈새에 어두운 적흑색 폴리곤(Polygon)을 렌더링하고 부드럽게 합성하여 자연스럽게 '입속'이 보이도록 실제 애니메이션 기법(2D VTuber 방식)을 적용했습니다.

### 4. 오디오 볼륨 스무딩 적용 (오작동 방지)
- 노이즈(작은 소음)에 입이 떨리지 않도록 Threshold를 0.015로 설정했습니다.
- 볼륨 변화를 컨볼루션(Convolution) 연산으로 부드럽게 만들어, 말이 연속될 때 입이 자연스럽게 오르락내리락 하도록 포물선(Envelope)을 개선했습니다.

### 4. 의존성 업데이트
- `requirements.txt`에 호환성을 검증한 `mediapipe` 라이브러리를 추가하고 설치를 완료했습니다. 

## 테스트 결과
내부 테스트용 스크립트(`test_local.py`)를 통해 가상의 이미지와 오디오로 영상 합성이 에러 없이, 빠르고 정상적으로 끝나는 것을 확인했습니다. 이제 프론트엔드에서 버튼을 눌러보시면 훨씬 퀄리티가 높아진 립싱크 영상을 보실 수 있습니다!
